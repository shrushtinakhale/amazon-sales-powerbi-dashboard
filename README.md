# Amazon Sales Performance Dashboard

## Overview
This project analyzes Amazon sales data using Power BI to provide insights into revenue, profitability, orders, and payment methods.

## Key Metrics
- Total Revenue
- Net Profit
- Total Orders
- Profit Margin (%)

## Dashboard Features
- KPI cards for high-level performance
- Trend analysis over time
- Category-wise sales breakdown
- Payment method share
- Top states by revenue contribution
- Interactive filters (Date, Category, Country)
